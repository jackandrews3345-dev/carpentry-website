Setup Guide for New Business Owner

Steps to customize this website for your business:

  1. Update Business Information: Change the business name, phone, and email above
  2. Change Login Credentials: Set a new username and password for admin access
  3. Add Your Social Media: Update your social media profiles
  4. Custom Domain (Optional): Set up your own domain name and update the website URL
  5. Export Settings: Download your settings as backup

Note: Changes are saved locally. For a custom domain, you'll need to redeploy the website.
